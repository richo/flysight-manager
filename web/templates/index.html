This wizard can help you generate a config file for the flysight-manager tool.

The flow will ask you to download 

flysight-manager.ini
. Save this to wherever you will run 
flysight-manager
from. You will need to edit this file to set you 
mountpoint

click here to connect to your dropbox account.

click here to connect to your youtube account.

flysight-manager will only have access to the app folder it creates (in /Apps), not anything else in your Dropbox. This is the folder that the files taken off your flysight will go into.
